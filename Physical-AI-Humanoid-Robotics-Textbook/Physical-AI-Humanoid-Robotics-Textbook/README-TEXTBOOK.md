# Physical AI & Humanoid Robotics Textbook

## Project Overview

This repository contains the **Physical AI & Humanoid Robotics Textbook** - a comprehensive educational resource covering Physical AI, humanoid robotics, ROS 2, digital twins, and Vision-Language-Action systems. The textbook is built using Docusaurus and follows academic standards with proper citations and peer-reviewed sources.

## Textbook Structure

The textbook is organized into the following modules:

### Core Modules
1. **Physical AI Foundations** - Core concepts of Physical AI and embodied intelligence
2. **ROS 2 Nervous System** - Communication architecture for robotics applications
3. **Digital Twin Simulation** - Gazebo and Unity workflows for robotics
4. **AI-Robot Brain** - NVIDIA Isaac for intelligent robot control
5. **Vision-Language-Action Systems** - Cognitive planning and interaction

### Supporting Content
- **Laboratory Exercises** - Hands-on exercises for each module
- **Capstone Project** - Voice-to-action humanoid robot project
- **Reference Materials** - Comprehensive bibliography and citation guidelines
- **Tutorials** - Setup and configuration guides
